# 🌐 Implémentation et Analyse de l'Architecture MPTCP (Multipath TCP)

> 🚧 **Statut du projet :** En cours de réalisation (Work in Progress)

Bienvenue sur le répertoire de mon projet de Master 1 en Réseaux et Télécommunications (Université Paris-Saclay).

## 🎯 Objectif du Projet
Ce projet vise à configurer, analyser et automatiser des connexions **Multipath TCP (MPTCP)** au sein d'un environnement Linux. L'objectif est d'étudier et de démontrer les avantages de cette architecture réseau, notamment :
* L'agrégation de bande passante pour optimiser le débit.
* La redondance et la résilience des liens (basculement en cas de panne).
* L'évaluation des indicateurs de performance réseau.

## 🛠️ Technologies et Outils
* **Système d'exploitation :** Environnement Linux (Configuration système et paramètres du noyau).
* **Réseaux & Sécurité :** Protocoles TCP/IP, MPTCP, analyse approfondie des paquets avec Wireshark.
* **Scripting & Automatisation :** Python, Bash/Shell.

## 📅 Livrables à venir
Le rapport d'étude complet ainsi que le code source seront publiés sur ce dépôt à la fin du projet. Les éléments suivants seront mis à disposition :
- [ ] Scripts Bash de configuration de l'environnement de test MPTCP.
- [ ] Scripts Python pour l'automatisation des tests de connectivité et de performance.
- [ ] Captures de trafic Wireshark (fichiers `.pcap`) pour l'analyse des trames.
- [ ] Rapport technique détaillé et documenté.
